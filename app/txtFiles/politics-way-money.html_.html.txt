P*TIME
OLTP
P*TIME ("Parallel* Transact-In-Memory Engine") is an in-memory relational database management system designed for real-time acquisition and analysis of update-intensive stream workloads.
http://dl.acm.org/citation.cfm?id=1316778
History
P*TIME was acquired by SAP in 2005. It was then integrated as the transactional catalog system in SAP HANA.
http://www.scmfocus.com/saphana/2017/08/03/hasso-plattner-students-invent-hana/
Data Model
Relational
Logging
Physical Logging
P*TIME uses a fine-grain differential logging technique to allow for fast recovery of the entire database in memory.
https://15721.courses.cs.cmu.edu/spring2018/papers/12-logging/lee-icde2001.pdf

In-Memory
