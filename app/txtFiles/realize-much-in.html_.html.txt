Truviso
Streaming
Truviso is a continuous query DBMS system built on Postgres. Truviso supports "always-on" queries for analyzing streaming data. This enables it to handle continuously flowing data instead of first accumulating data and then running batch queries for reporting and analysis.
History
Truviso was acquired by Cisco in May 2012
